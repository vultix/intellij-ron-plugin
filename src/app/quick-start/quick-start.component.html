Quick Start Guide

Getting the Plugin

IntelliJ-RON supports any of JetBrain's IDEs. If you don't know where to start, we'd recommend IntelliJ Idea Ultimate or Clion.

IntelliJ-RON is dependent upon the IntelliJ Rust plugin. To install it, follow their quickstart guide.

To install the RON plugin, go to Settings / Preferences -> Plugins -> Marketplace and search for "RON". Install the plugin titled "RON (Rusty Object Notation)" to begin.

Using the Plugin

As RON is built to represent Rust types, open or create a new Rust project. Then, create a new RON file.

See Features to learn how to import a Rust type.
