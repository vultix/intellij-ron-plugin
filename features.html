IntelliJ-RON Plugin

Features

Current keybindings: Windows / Linux Default

Importing Ron Types

IntelliJ-RON has a custom syntax for importing Rust types. This enables code navigation, completion and validation.

To import a type:

  • Create an 'import' block comment at the top of your file by typing /*!.
  • Inside the 'import' block comment, simply type the name of any Rust struct, enum, or type alias.

Syntax Highlighting

IntelliJ-RON adds support for completely customizable lexer-based highlighting. All color schemes are customizable and we support both Default and Darcula themes out of the box.

Code Formatter

Reformat CodeCtrl+Alt+L re-formats the current file or selection using our custom, functional, fully-customizable formatter based on IntelliJ's formatting engine.

Code Selection

To select a block of code place a cursor inside block and press Ctrl+W to expand it to an expression, to a line, to a logical block of code, and so on. To shrink selection pressCtrl+Shift+W as many times as needed.

Code Commenting

After selecting block, you can apply a block comment (Ctrl+Shift+/) or line comment (Ctrl+/) to it. The same shortcuts will remove existing comments from selected lines too.

Code Folding

Code folding, auto-insertion of braces, brackets & quotes, matching brace/bracket highlighting, etc...

On-the-fly analysis

Errors are shown as you type. Among others, we catch syntax errors, missing struct fields, and invalid types. We also support IntelliJ's built-in spell-checker.

Parameter Hints

In the editor, use Show Parameter HintsCtrl+P when editing enums or tuples and you can see type hints.

Type Information

Use Ctrl+Shift+P to view the type of the selected struct, enum, map, list, etc...

Quick Documentation

Use Quick DocumentationCtrl+Q to show the docs for the symbol at caret.
